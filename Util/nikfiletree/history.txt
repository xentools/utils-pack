Программа NikFileTree - Версия 1.8 - История версий

Обозначения:
  - Исправлена ошибка
  + Новая возможность
  * Косметические доработки

----------------------------------------------------------------------------
Версия 1.8 <- 1.7
  + Количество знаков после запятой в размере теперь можно задавать
  - Исправлена ошибка, возникающая при запуске программы, если в реестре
    не было настроек. А так и бывает при первой инсталляции программы :(
  - При неудачной попытке открытия css-файла (для HTML) потом невозможно
    было открыть файл-результат, даже если правильно задать css-файл
  - Исправлена ошибка в работе с именами файлов в кавычках
  * Улучшен по структуре получаемый HTML-результат, изменился css-файл
  * Изменился интерфейс: На вкладке Дополнительно теперь почти все контролы
    из переключателей переделаны на выпадающие списки
  * Уточнены сообщения об ошибках

----------------------------------------------------------------------------
Версия 1.7 <- 1.6
  + Появилась отключаемая сортировка! По названиям, размерам, времени
    создания/модификации (по возрастанию и убыванию)
  + Появился еще один вид результата: HTML-страница!
  + Дополнительные настройки для HTML
  + В заголовке окна программы показывается этап построения дерева
    (Сканирование, Вывод в файл)
  + Можно пропускать пустые папки, даже если выводятся только папки
  + Теперь возможно хранить несколько различных настроек и выбирать любую
    из них для каждого конкретного случая
  - При неудачной попытке открытия файла-результата на запись блокировались
    все контролы кроме кнопок Построить дерево, Открыть результат и т.п.
  - Если выводились только папки, то не считался размер папок.
  - Маска файлов теперь доступна всегда, даже если выводятся только папки
  * Размеры файлов выводятся как числа, без указания единицы измерения
    (она вынесена в строку с заголовками)
  * Убрано цветовое выделение настроек. Иначе скоро будет слишком пестро :)
  * Уточнены сообщения об ошибках

----------------------------------------------------------------------------
Версия 1.6 <- 1.5
  + Фильтр на файлы для вывода по нескольким маскам сразу: *.zip,*.rar и т.п.
  + Подсчет размеров папок (суммарные размеры файлов по маске с подпапками)
  + Можно не выводить пустые папки (удобно для поиска файлов по маске)
  + Можно не выводить в плоском списке стартовую папку
  + Можно выводить в плоском списке относительные пути к файлам/папкам
  + Можно выводить в плоском списке атрибуты файлов/папок
  + Задаваемый пользователем разделитель для плоского списка
  * Вместо папки по-умолчанию для файла результата - галочка "Не изменять"
    расположение и имя файла-результата
  + Стартовая папка запоминается, но параметр командной строки главнее
  - Убран лишний пробел перед кавычками в имени в плоском списке для Dos
  * Цветом выделяются настройки, предназначенные только для дерева (зеленый)
    и только для плоского списка (бирюзовый)

Версия 1.5 <- 1.4
  - Теперь корректно обрабатывается имя файла-результата в кавычках.
  * Параметры разнесены на разные вкладки
  + Для плоского списка кроме названий предусмотрен настраиваемый вывод:
      размеров файлов
      даты/времени создания
      даты/времени изменения
  + Размеры файлов можно выводить в байтах/килобайтах/мегабайтах
  + Настраиваемый формат вывода даты/времени
  + Вместо нулевого размера у папок можно выводить какое-нибудь слово,
    например, ПАПКА
  + Для плоского списка можно вывести строку с заголовками
  + Файл-результат исключается из себя самого
  + Названия файлов/папок в плоском списке можно заключать в кавычки:
      никогда
      только, если имя файла содержит пробелы, запятую или точку с запятой
      всегда
  + Можно задать папку по-умолчанию для файла результата и она не будет
    меняться при изменении стартовой папки
  + Сохранение настроек в реестре и их восстановление

Версия 1.4 <- 1.3
  + Возможность пропускать скрытые и системные файлы и папки
  - Снова работает CheckBox "Выводить названия файлов" (в версии 1.3 файлы
    выводились всегда)
  - При вызове через контекстное меню Проводника теперь правильно передается
    стартовая папка
  * Косметические доработки

Версия 1.3 <- 1.2
  - Исправлена ошибка вывода стартовой папки всегда в Win-кодировке
  + Добавлена возможность выводить дерево псевдографикой в Dos-кодировке
  + Добавлена возможность интегрироваться в Проводник и удаляться из него
  * Кнопки "Построить дерево" и "Прервать" совмещены
  + Кнопка "Открыть результат" недоступна пока строится дерево
  + Сначала выводятся файлы, а потом папки, их файлы и подпапки и т.д.
  + При изменении стартовой папки автоматически меняется путь к файлу-
    результату
  + Появилась программа установки, но можно обойтись и без нее
  + Сделано окно About
  + Появился Help

Версия 1.2 <- 1.1
  + Добавлена возможность выводить только файлы (в плоском списке)

Версия 1.1 <- 1.0
  + Добавлен вывод имен файлов (отключаемый)
  + Символ, обозначающий файл, можно поменять
  + Добавлен вид результата - дерево или плоский список
  + Возможность выводить все папки заглавными буквами, а все файлы строчными
  * Программа переименована из NikDirTree в NikFileTree

Версия 1.0
  + Вывод дерева папок в Dos или Windows кодировках
  ...
