1. DESCRIPTION

The BDV Notepad is a more intuitive text editor than the standard issued Windows Notepad. By using the BVD Notepad you will be able to have more functions, easy-to-use and easy to understand interface. All of these functions for a grand price of zero! How lucky you are to use this programme for that big price!

2. INSTALL/UNINSTALL

INSTALL: The BDV Notepad does not require any installations whatsoever, and it could be run as is, simply by double-clicking on its icon. You may copy the "Notepad.exe" file that came with your download, to the original Windows folder on your hard drive, and replace the standard Notepad that is installed with the your Windows operating system. For your convenience, we recommend that you also copy the language, and the codepage files as well.
IMPORTANT NOTICE: Should you decide to replace the standard Windows Notepad, we strongly recommend that you make a back-up copy of the original Windows Notepad first! Furthermore, the BVD Notepad will use the Windows registry only if you decide to use the "Associations" option thereof.

UNINSTALL: You may simply delete the BVD Notepad from your computer should you decide not to use it any longer.
IMPORTANT NOTICE: If you did use the "Associations" option, and wished to delete the BVD Notepad from your computer, you must first un-check all checked items on the "Associations" tab where you had checked previously. Then the defualts of the "Associations" shall be re-stored automatically upon your next re-boot of your computer. Furthermore, it is necessary that the "Notepad.ini" file be deleted as well in order to have the full default values get en-abled automatically by your operating system.

3. HOW TO USE

Essentially, this programme does not differ any more than its counter part that ships with the Windows - ie, the original Windows Notepad. However, please, make a note of the following new features, and/or omissions as per list below

Omissions:

  • No page headers, and
  • No other files except text files can be opened.

    New features/additions:

  • Intuitive, and easy to configure interface (Improved!);
  • Auto save;
  • Character filters;
  • Insert characters/colours/file names/etc... (Improved!);
  • Edit text files of any size;
  • Shortcuts;
  • Unlimited Undo/Redo *;
  • Improved printing engine;
  • Extended find-and-replace options *;
  • Multi-lingual interface, editing or creating new language files;
  • Win-DOS-KOI8 codepage convertion with the user code-pages support;
  • URL autodetect and start *;
  • Case changes (Improved!);
  • Display of special characters (New!).

    * Run-time requirements:
    No less than the RichEdit 2.0 library is required for the above functions to operate properly. However, this library is included in all versions of the Windows - except the Windows 95. If necessary, the RichEdit 3.0 (324 Kb) library may be loaded from www.badevlad.hotmail.ru.
